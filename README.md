## Description
A small collection of simple algorithms (currently only sorting algorithms) programmed in Java 8.

Currently includes:  
  `Quicksort`  
  `Bubblesort`  
  `Heapsort`  
  `Insertionsort`    
  `Selectionsort`     
           
        
Also includes a class to test and time sorting algorithms.
